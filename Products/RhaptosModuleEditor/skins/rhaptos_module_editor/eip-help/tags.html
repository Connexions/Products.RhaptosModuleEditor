Complete CNXML tag list

This page contains a list of tags used to markup content in CNXML. The block tags are used to set up the structure of the document. Inline tags are used to markup parts of the text within the block tags. Some tags (i.e., code and note) contain the display attribute, which specifies whether the tag should be displayed as block or inline. These tags are noted (†). CALS table tags are used to create tables and media tags insert multimedia into CNXML documents.

See the individual tag pages for more information and examples.

† The default display behavior can be changed using the display attribute. If this attribute is set to "inline", then the tag will be included in the current line of text; if the display attribute is set to "block", then the element will be set apart from the surrounding text.
