Description

The space tag is used to create inline space within other elements, such as para.

Example

This CNXML:


<exercise id="chem-exercise"> 
  <problem id="chem-problem-1">
    <para id="chem-para1">
      Fill in the blanks:
    </para>
    <para id="chem-para2"> 
      A water molecule is made up of two 
      <space count="10" effect="underline"/>
      atoms and one
      <space count="10" effect="underline"/>
      atom.
    </para>
  </problem>
  <solution id="chem-solution-1">
    <para id="solution1"> Hydrogen; Oxygen </para>
  </solution>
</exercise>

Results in this display:

Exercise 1

Fill in the blanks:
A water molecule is made up of two
atoms and one
atom.

Solution

Hydrogen; Oxygen

Attributes(what's this?)

  • count (optional): The number of spaces to be inserted (default is "1"). Must be a positive integer.
  • effect (optional): The presentation style of the blank space.
    Possible values:
    • normal - Blank space is inserted (default).
    • underline - An underlined space is inserted.
  • id (optional): A unique identifier, whose value must begin with a letter and contain only letters, numbers, hyphens, underscores, colons, and/or periods (no spaces).

Children(what's this?)

The space tag is an empty tag and does not contain any children.

Parents(what's this?)

preformat, para, title, label, cite, cite-title, link, emphasis, term, sub, sup, quote, foreign, footnote, note, item, code, caption, commentary, meaning, entry
